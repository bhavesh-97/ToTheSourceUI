GSAP Rule Guide: Rules define animations for specific elements.
  • Type: fromTo (animate from A to B), to (animate to target), from (animate from target), set (set values instantly), timeline (sequence multiple steps)
  • Selector: CSS selector like .classname, #id, div, or .parent .child
  • From/To: JSON with GSAP properties - opacity, x, y, scale, rotation, rotationX, rotationY, skewX, skewY, backgroundColor, borderRadius, filter, clipPath, transformOrigin
  • Duration, Ease, Stagger, Delay are inherited from Global defaults - only set in rule if you want to override

GSAP Callback Guide: Callbacks execute custom code at specific animation events.
  • Name/Event: Use one of: onStart (animation begins), onUpdate (every frame), onComplete (animation finishes), onRepeat (each repeat), onReverseComplete (reverse finishes)
  • Script: JavaScript code. Available variables: tween (the GSAP tween), element (animated DOM element)
