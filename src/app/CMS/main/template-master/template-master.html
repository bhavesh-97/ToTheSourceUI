@if (showTemplateGuide) {
Simple Variable
{{variableName}}

Replaces with the value of variableName from your data source. Works in HTML content and attributes.

HTML Content <h1>{{title}}</h1> Image Attribute <img src="{{heroImage}}" alt="{{title}}">
Fallback Value
{{variableName | Default Value}}

Uses "Default Value" if the field is empty, null, or missing from the data source.

Array Loop
{{#each items}} ... {{/each}}

Iterates through an array. Each item's properties are available as variables inside the loop.

Example <div class="services">{{#each services}}<div class="card"><h3>{{name}}</h3><p>{{description}}</p></div>{{/each}}</div>
Conditional Rendering
{{#if condition}} ... {{else}} ... {{/if}}

Shows content only if the condition is truthy. Use {{else}} for fallback content.

Example {{#if showBanner}}<div class="banner">{{bannerText}}</div>{{else}}<div class="banner">Default Banner</div>{{/if}}
Data Source Mapping
Manual Entry

Add fields in the Dynamic Data Source panel. The Key name becomes the variable name.

Key: "title" → {{title}}
API Endpoint

The API response keys become variables. Click Test Connection to see available keys.

Response: name → {{name}}
Pro Tips
  • Use {{item.title}} for nested object properties
  • Variables work inside HTML attributes: href="{{linkUrl}}"
  • Combine loops with conditionals for dynamic layouts
  • Always set fallback values for optional fields
  • Test your API connection to see the exact response structure
